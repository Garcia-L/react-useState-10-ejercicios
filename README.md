# ⚛️ React useState – 10 Ejercicios Progresivos

Este repositorio contiene una serie de 10 ejercicios prácticos enfocados en el uso del hook `useState` en React.

El objetivo es aprender el uso básico de useState.

---

## 🚀 Tecnologías utilizadas

> No se utiliza Vite, Create React App ni herramientas de build.  
> El enfoque es comprender React desde lo esencial.

---

## 🧠 Objetivo de aprendizaje

- Comprender cómo funciona `useState`
- Manejar estado en diferentes estructuras:
  - Primitivos (number, string, boolean)
  - Arrays
  - Objetos
- Construir lógica interactiva en la UI

---

## 📂 Ejercicios incluidos

1. Contador básico  
2. Contador con reset  
3. Contador con valor inicial dinámico  
4. Contador con límites  
5. Input controlado  
6. Formulario simple  
7. Lista dinámica  
8. Lista con eliminación  
9. Lista con edición  
10. Mini carrito de compras  

---

## 💡 Metodología

Cada ejercicio fue desarrollado con el objetivo de:

- Practicar lógica
- Entender cómo React reacciona a cambios de estado
